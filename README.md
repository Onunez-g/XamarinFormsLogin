# XamarinFormsLogin
Xamarin Forms basic App. Working as first starter in Xamarin Forms.

## Contents

Right now the App currently has:

* Basic Login Page
* Basic Sign up Page
* Tabbed Page with 5 basic Pages
* Discovery Page with a List of card views
